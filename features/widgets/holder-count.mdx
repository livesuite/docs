# Holder Count Widget

Display real-time holder count and growth trends. Show token adoption and community growth as it happens.

## Configuration

**Required:**
- **Token Address (mintId)** - Your token's Solana mint address

**Optional:**
- **Goal Amount** - Optional holder milestone to track (e.g., 1000 holders)
- **Accent Color** - Customizable accent color

## Widget URL

**Basic:**
```
https://livesuite.app/widgets/holder-count/YOUR_MINT_ID
```

**With Goal Tracking:**
```
https://livesuite.app/widgets/holder-count/YOUR_MINT_ID?goalAmount=1000&accentColor=#00ff00
```

## Features

- Real-time holder count updates
- Optional goal tracking for milestones
- Holder growth indicators
- Historical data integration
- Clean numeric display

## Use Case

Show token adoption and community growth in real-time. Perfect for highlighting when you hit holder milestones.

**Recommended Setup:**
- Position: Bottom overlay area
- Size: Small (300-400px width)
- Great for community-focused streams
