All-in-one widget showing comprehensive token statistics. Perfect compact display for showing everything at a glance.

## Configuration

**Required:**
- **Token Address (mintId)** - Your token's Solana mint address

**Optional:**
- **Accent Color** - Customizable accent color
- **Display Options** - Configurable stat visibility

## Widget URL

**Basic:**
```
https://livesuite.app/widgets/token-stats/YOUR_MINT_ID
```

**With Custom Color:**
```
https://livesuite.app/widgets/token-stats/YOUR_MINT_ID?accentColor=#ff0088
```

## Features

- Multiple metrics in single widget
- Current viewers count
- Market cap display
- Price tracking
- Streaming time
- All key metrics in one place
- Configurable display options

## Use Case

Perfect when you want to show all key stats in one compact widget. Great for streams where space is limited.
