# Live Chat Widget

Display live chat messages from your stream directly in OBS. Show viewer engagement by displaying chat messages on stream.

## Configuration

**Required:**
- **Token Address (mintId)** - Your token's Solana mint address

**Optional:**
- **Message Limits** - Configurable number of messages to display
- **Styling** - Customizable chat appearance

## Widget URL

```
https://livesuite.app/widgets/live-chat/YOUR_MINT_ID
```

## Features

- Real-time chat message display
- Message formatting and styling
- Configurable message limits
- Clean chat overlay design
- Automatic message updates

## Use Case

Great for highlighting community interaction and showing active engagement. Display recent chat messages to show your community is active and engaged.
