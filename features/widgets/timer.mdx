Track how long you've been streaming with a clean, formatted timer display.

## Configuration

**Required:**
- **Token Address (mintId)** - Your token's Solana mint address

**Optional:**
- **Accent Color** - Customizable color (default: `#00ff88`)

## Widget URL

**Basic:**
```
https://livesuite.app/widgets/obs-timer/YOUR_MINT_ID
```

**With Custom Color:**
```
https://livesuite.app/widgets/obs-timer/YOUR_MINT_ID?accentColor=#ff00ff
```

## Features

- Real-time streaming duration calculation
- Formatted display (hours:minutes:seconds)
- Automatic updates every few seconds
- Clean, minimal design perfect for overlays
- Custom accent color support

## Use Case

Show your audience exactly how long you've been streaming a token. Great for all streams.

**Recommended Setup:**
- Position: Top-left or bottom-left corner
- Size: Medium (400-600px width)
- Perfect for showing streaming dedication
