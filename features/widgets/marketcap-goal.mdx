# Market Cap Goal Widget

Visualize progress toward a market cap milestone with a progress bar. Build anticipation by showing how close you are to hitting major milestones.

## Configuration

**Required:**
- **Token Address (mintId)** - Your token's Solana mint address
- **Goal Amount (USD)** - Target market cap in USD

**Optional:**
- **Accent Color** - Customizable accent color

## Widget URL

**Basic:**
```
https://livesuite.app/widgets/marketcap-goal/YOUR_MINT_ID/1000000
```

**With Custom Color:**
```
https://livesuite.app/widgets/marketcap-goal/YOUR_MINT_ID/1000000?accentColor=#00ffff
```

## Features

- Progress bar showing completion percentage
- Current vs. goal market cap comparison
- Visual progress indicator
- Real-time market cap tracking
- Percentage completion display

## Use Case

Perfect for building hype when approaching major milestones like $100K, $500K, $1M, etc.

**Recommended Setup:**
- Position: Near token info or center of screen
- Size: Medium to large (500-800px width)
- Use when approaching milestones
