# User Dashboard

Your personalized control center for managing widgets, chat moderation, and tracking your engagement on LiveSuite. Access your dashboard at `livesuite.app/dashboard` after connecting your wallet.

## Dashboard Overview

The User Dashboard provides centralized access to all your personalized features and tools. Everything is organized in clear sections with easy navigation.

**Authentication Required:**
- Connect your wallet to access
- Your data stays private and secure

## Dashboard Sections

### Widgets

Manage all your saved OBS widgets in one place.

**Widget Management:**
- View all your saved widgets
- See widget type, name, and creation date
- Track usage counts and last accessed times
- Quick actions: Edit, Duplicate, Delete

**Widget Cards Display:**
- Widget name and type badge
- Widget icon (type-specific)
- Creation date and last used date
- Usage count
- Public/private status indicator

**Quick Actions:**
- **Edit** - Open widget in visualizer with settings pre-loaded
- **Duplicate** - Create a copy for easy variations
- **Delete** - Remove widgets you no longer need

**Stats Bar:**
- Total widgets count
- Quick overview of your widget collection

### Chat Bot

Automated chat moderation system for your pump.fun streams.

**Token Selection:**
- Choose which token to moderate
- Dropdown with your verified tokens
- Integration with your favorites and widgets
- Batch verification for multiple tokens

**Configuration:**
- Custom word filters (whole word matching)
- Regex filters (pattern-based matching)
- Case sensitivity options
- Action types: Ban, Delete Message, Warning
- Warning thresholds
- Custom commands (e.g., `!website`, `!discord`)

**Monitoring:**
- Real-time log display
- Action tracking (bans, deletions, warnings)
- Statistics dashboard
- Command execution logs

**Setup Process:**
1. Select token to moderate
2. Verify ownership via chat message
3. Configure filters and commands
4. Set warning thresholds
5. Monitor activity through logs

See [Chat Moderation Bot](/features/tools/chat-moderation) in Tools for more details.

### Platform Engagement

Track your voting activity and community participation.

**Voting Statistics:**
- Total votes cast (up and down)
- Vote distribution breakdown
- Most voted tokens
- Voting activity timeline

**Voting History:**
- Paginated vote history
- Token information for each vote
- Vote type (up/down)
- Timestamps
- Quick navigation to token pages

**Community Metrics:**
- Engagement scoring
- Community participation trends
- Token popularity tracking

## Getting Started

1. **Connect Your Wallet** - Click "Sign In" and connect your wallet
2. **Verify Token Ownership** - Verify tokens you want to moderate (if using chat bot)
3. **Create Widgets** - Use the Widget Visualizer to create and save widgets
4. **Set Up Moderation** - Configure chat bot for your streams (optional)
5. **Track Engagement** - View your voting history and statistics
