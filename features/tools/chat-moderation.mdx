Automated chat moderation system for your pump.fun streams. Access from your dashboard to set up custom filters, commands, and automated actions.

## Features

**Custom Filters:**
- Word filters (whole word matching)
- Regex filters (pattern-based matching)
- Case sensitivity options

**Automated Actions:**
- Ban users automatically
- Delete messages automatically
- Issue warnings with configurable thresholds

**Warning System:**
- Configurable warning thresholds
- Automatic action when threshold is reached
- Warning accumulation tracking

**Custom Commands:**
- Create commands like `!website` or `!discord`
- Set custom response text
- Enable/disable commands as needed

**Real-Time Monitoring:**
- Live log display
- Action tracking (bans, deletions, warnings)
- Statistics dashboard
- Command execution logs

## Setup

1. Verify token ownership via chat message
2. Configure filters and commands in dashboard
3. Set warning thresholds and actions
4. Monitor activity through real-time logs

## Use Cases

- Automatically moderate spam and toxic messages
- Provide quick links via custom commands
- Maintain stream quality without constant monitoring
- Track moderation statistics and activity

<Note>
  Chat moderation requires token ownership verification and is accessible from the User Dashboard.
</Note>
