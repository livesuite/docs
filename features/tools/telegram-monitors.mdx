# Telegram Monitors

Set up automated Telegram alerts for important events. Never miss new token launches or trending activity.

## Monitor Types

**New Tokens:**
Get alerts when tokens launch on pump.fun.

**Trending Tokens:**
Notifications for trending activity and momentum shifts.

**High Follower Streamers:**
Alerts for streamers with large Twitter followings joining the platform.

## Features

- Configurable thresholds
- Customizable alert frequency
- User subscription management
- Real-time monitoring

## Use Cases

- Never miss a new token launch
- Get early alerts on trending opportunities
- Track high-influence streamers
- Stay ahead of the market

## Setup

1. Join the LiveSuite Telegram channel
2. Configure your alert preferences
3. Set thresholds for each monitor type
4. Receive alerts in real-time
