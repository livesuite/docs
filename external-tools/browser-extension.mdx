A browser extension that brings LiveSuite token scanning and data directly to your web browsing experience. Scan tokens, view real-time data, and access LiveSuite features without leaving your current page.

## Features

- Quick token scanning from any webpage
- Real-time token data overlay
- Streamer token information at your fingertips
- Price, market cap, and volume data
- Live streaming status and viewer counts
- Direct links to LiveSuite dashboard
- Seamless browser integration
