A Telegram bot that scans streamer tokens and displays comprehensive token information including streaming stats, market data, and more.

## Features

- Token scanning by streamer tokens addresses (paste address or use `/scan`)
- Comprehensive token data display (price, market cap, volume, supply, holders, streaming stats)
- Trending tokens discovery
- Token search functionality
- Favorites system (requires account linking)
- Notification system for group/channel administrators
- Statistics and performance metrics

## Commands

- `/start` - Start the bot and get instructions
- `/help` - Show detailed help and examples
- `/stats` - Show bot statistics and performance metrics
- `/trending` - Show currently trending tokens
- `/search <query>` - Search tokens by name, symbol, or mint ID
- `/favorites` - Show your favorite tokens (requires linked account)
- `/notify` - Manage token notifications for this chat (admin only)

## Token Scanning

- Paste any Solana contract address in chat to scan
- Or use `/scan` command with address
- Automatic detection of valid Solana addresses
- Loading messages during scan
- Error handling for invalid addresses or API failures

## Token Data Displayed

- Token name, symbol, and description
- Price, market cap, and volume information
- Supply and holder count
- Live streaming status and viewer count
- Peak viewer metrics and session history
- Social media links
- Direct links to LiveSuite dashboard
- Token images (when available)

## Interactive Features

- Refresh button for token data
- Favorite/Unfavorite buttons (when user account is linked)
- Inline keyboard buttons for trending/search results
- Inline buttons to scan tokens from lists
- Markdown formatting for messages
- Photo messages for token images
