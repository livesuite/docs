# Discord Scanner Bot

A Discord bot that scans streamer tokens and displays comprehensive token information including streaming stats, market data, and more.

## Features

- Token scanning by streamer tokens address via `/scan` command
- Comprehensive token data display using Discord embeds
- Works in both guilds (servers) and private DMs
- Trending tokens discovery
- Token search functionality
- Favorites system (requires account linking)
- Notification system for server administrators
- Statistics and performance metrics

## Commands

- `/scan mint_id:<address>` - Scan a token by streamer token address
- `/help` - Show detailed help and command examples
- `/stats` - Show bot statistics and performance metrics
- `/trending [limit:<number>]` - Show currently trending tokens
- `/search query:<search_term>` - Search tokens by name, symbol, or mint ID
- `/favorites` - Show your favorite tokens (requires linked account)
- `/notify` - Manage Discord notifications for tokens in this server (admin only)

## Token Data Displayed

- Token name, symbol, and description
- Price, market cap, and volume information
- Supply and holder count
- Live streaming status and viewer count
- Peak viewer metrics and session history
- Social media links
- Direct links to LiveSuite dashboard
- Token images (when available)

## Interactive Features

- Refresh button for token data
- Favorite/Unfavorite buttons (when user account is linked)
- Inline buttons for trending/search results to scan tokens
- Discord embeds for rich formatting
- Discord button components for interactions
