## Documentation

This repository contains the documentation for LiveSuite features, widgets, analytics, and tools.

Built with [Mintlify](https://mintlify.com).

## Local Development

Install the Mintlify CLI:

```bash
npm i -g mint
```

Run locally:

```bash
mint dev
```

View at `http://localhost:3000`.
